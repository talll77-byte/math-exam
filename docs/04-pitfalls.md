# מוקשים חבויים

## גזירה

| מוקש | טעות | תיקון |
|------|------|-------|
| כפל חזקות | $x^4\cdot x=x^4$ | $x^4\cdot x=x^5$ |
| נגזרת פנימית | $\frac{d}{dx}(6-x)=1$ | $=-1$ |
| מנה קבוע | שכחת מינוס בנוסחת קיצור | $\left(\frac{c}{g}\right)'=\frac{-cg'}{g^2}$ |
| $f''=0$ | מסיקים "פיתול" אוטומטית | בדוק סימן $f'$ לפני ואחרי — חובה טבלה |
| $f''$ לבד | מנסים למצוא עלייה/ירידה | רק טבלת $f'$ |
