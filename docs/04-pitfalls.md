# מוקשים — לפני המבחן

## גזירה

| מוקש | טעות נפוצה | תיקון |
|------|------------|----------|
| כפל חזקות | מחברים מעריכים במקום מכפילים | $x^4\cdot x=x^5$ |
| נגזרת פנימית | שוכחים מינוס | $\frac{d}{dx}(6-x)=-1$ |
| מנה קבוע | שוכחים מינוס בנוסחה | $\left(\frac{c}{g}\right)'=\frac{-cg'}{g^2}$ |
| נגזרת שנייה אפס | מסיקים פיתול בלי בדיקה | בדוק סימן הנגזרת הראשונה |
| נגזרת שנייה לבד | מחפשים עלייה וירידה | רק טבלת הנגזרת הראשונה |

---

## תחום הגדרה

| מוקש | טעות נפוצה | תיקון |
|------|------------|----------|
| שורש במכנה | משתמשים ב-greater-or-equal | $g(x)>0$ בלבד |
| דלתא שלילית במכנה | מחפשים אפסים | אין אילוץ — "כל x" |
| מכנה + שורש | פותרים יחד | הפרד מכנה / שורש, חתוך בסוף |
| פרבולה בוכה | חיובי מחוץ לשורשים | חיובי **בין** השורשים |

---

## טעויות נפוצות בחומר לימוד

!!! warning "תרגיל 12 — תחום שורשים"

    לפעמים מופיע בטעות:

    <div class="formula-block" markdown="1">

    $$1\le x\le 3$$

    </div>

    **התשובה הנכונה:**

    <div class="formula-block" markdown="1">

    $$-1\le x\le 3$$

    </div>

!!! warning "חזקה זוגית"

    <div class="formula-block" markdown="1">

    $$x^4=1 \;\Rightarrow\; x=1 \;\|\; x=-1$$

    </div>

    שני פתרונות — אל תפספס את השלילי.

!!! warning "נגזרת שנייה"

    נגזרת שנייה חיובית → מינימום. **לא** אומרת שהפונקציה עולה בכל האזור.
